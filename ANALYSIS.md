## 🔧 **How to Use the Images**

1. Copy the generated images to your replica folder:
```bash
# Create images folder
cd replica-1
mkdir images

# Copy images (adjust paths as needed)
cp /path/to/couple_hero_image.png images/
cp /path/to/love_story_image.png images/
cp /path/to/gallery_photo_1.png images/
```

2. Update the HTML to use real images:
```html
<!-- Replace hero background in CSS -->
background-image: url('images/couple_hero_image.png');

<!-- Replace love story image -->
<img src="images/love_story_image.png" alt="Our love story">

<!-- Replace gallery placeholders -->
<img src="images/gallery_photo_1.png" alt="Gallery photo">
```
